# student_event_volunteer_system
Web-based Volunteer &amp; Event Management System untuk Organisasi Mahasiswa (Ormawa) yang dirancang untuk mengelola data volunteer, perencanaan event, pembagian tugas, dan monitoring kehadiran secara terpusat. Sistem ini membantu meningkatkan efisiensi koordinasi, transparansi informasi, serta pengelolaan kegiatan organisasi secara terstruktur.
